번역 자막
설정 가이드

TikTok LIVE Studio 음성을 실시간 번역해 자막으로 송출하는 방법을 영상으로 안내합니다.

형식 영상 난이도 입문

▶ 재생 버튼을 누르세요 · 전체화면 지원

요약

  1. 사이드바 → 번역 자막 이동.
  2. 표시 언어 선택 (한국어/영어/일본어/중국어 등).
  3. 폰트 크기/색상/그림자/외곽선 조절.
  4. 설정 저장 클릭으로 영구 저장.
  5. 오버레이 URL을 OBS 브라우저 소스에 추가.

참고

  • 음성 인식 — TikTok 내장 Web Speech API (API 키 불필요).
  • 번역 엔진 — Google / Papago / LibreTranslate 선택 가능.
  • 표시 시간 — 자막 노출 ms 단위 설정.
