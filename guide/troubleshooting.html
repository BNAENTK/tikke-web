자주 발생하는 문제와
해결 가이드

대부분의 문제는 한두 가지 원인에서 발생합니다. 이 글은 Tikke 사용 중 가장 자주 보고되는 10가지 상황의 원인과 해결 단계를 정리합니다. 본인 상황과 가장 가까운 항목을 골라 위에서 아래로 차례대로 시도해보세요.

예상 소요 6분 난이도 참고 업데이트 2026.05

1. "라이브 연결" 버튼을 눌러도 인증 창이 안 뜬다

주요 원인

방화벽이나 백신 프로그램이 Tikke의 외부 통신을 차단했을 가능성이 높습니다.

해결 절차

  1. Windows 보안 → 방화벽 → 앱 허용 목록에 Tikke가 있는지 확인. 없으면 추가.
  2. 알약·V3 같은 백신이 Tikke 실행 파일을 차단하지 않았는지 격리 목록 확인.
  3. VPN을 사용 중이라면 잠시 끄고 다시 시도. (일부 VPN이 TikTok 인증을 막습니다.)
  4. 회사·학교 등 공용 네트워크라면 모바일 핫스팟으로 한 번 테스트.

2. TikTok 라이브를 켰는데 Tikke가 감지하지 못한다

주요 원인

입력한 핸들이 잘못됐거나, TikTok 측에서 라이브 시작 신호가 평소보다 늦게 발행되고 있을 수 있습니다.

해결 절차

  1. 설정 → 계정에서 등록한 TikTok ID(@ 제외)가 정확한지 확인. 대소문자·점·언더스코어까지 일치해야 하며, @ 기호는 포함하면 안 됩니다.
  2. 라이브 시작 후 1~2분 정도 대기. TikTok 신호 발행에 최대 1분의 지연이 있을 수 있습니다.
  3. 여전히 감지 안 되면 Tikke의 "연결 새로고침" 버튼을 누르거나 앱을 재시작.
  4. 같은 시점에 컴퓨터에서 tiktok.com/@자신의핸들/live URL이 열리는지 확인. 안 열린다면 TikTok 측 일시적 문제일 수 있습니다.

3. 채팅이 한참 늦게 들어온다 (지연 5초 이상)

주요 원인

네트워크 환경이 불안정하거나, 같은 PC에서 다른 무거운 작업이 진행 중일 수 있습니다.

해결 절차

  1. 다른 다운로드·스트리밍을 잠시 중단해보세요.
  2. Wi-Fi라면 가능하면 유선으로 전환. Wi-Fi 신호 강도가 약하면 지연이 큽니다.
  3. Tikke 설정 → 진단에서 현재 핑(ping)을 확인. 300ms 이상이면 네트워크 문제입니다.
  4. OBS·게임 등 GPU를 많이 쓰는 앱과 함께 실행 중이라면 Tikke를 다른 PC로 분리하는 것을 고려.

4. 오버레이 URL을 LIVE Studio에 넣었는데 화면에 안 보인다

주요 원인

URL이 잘못 복사됐거나, LIVE Studio의 소스 크기·위치가 화면 밖으로 설정됐을 수 있습니다.

해결 절차

  1. URL을 브라우저에 직접 붙여넣어 위젯이 정상 표시되는지 확인. 표시되지 않으면 URL 자체 문제.
  2. LIVE Studio에서 해당 소스를 우클릭 → "변형 → 화면에 맞춤" 시도.
  3. 소스의 가시성(눈 아이콘) 토글 확인. 실수로 꺼져 있는 경우가 많습니다.
  4. Tikke에서 URL을 재발급(설정 → 오버레이 → 토큰 재발급)하고 LIVE Studio에서 교체.

5. 외국어 채팅이 번역되지 않는다

주요 원인

번역 기능이 꺼져 있거나, 메시지가 너무 짧아 언어 감지가 실패했을 수 있습니다.

해결 절차

  1. 설정 → 번역에서 "자동 번역" 토글이 켜져 있는지 확인.
  2. 도착 언어가 "한국어"로 설정되어 있는지 확인.
  3. 이모지나 한 단어 메시지는 번역되지 않습니다 (의도된 동작).
  4. 특정 언어에서만 안 된다면 "지원 언어 목록"에서 해당 언어가 활성화되어 있는지 확인.

6. TTS가 작동하지 않는다 (음성이 안 나옴)

주요 원인

운영체제의 오디오 출력 장치가 변경됐거나, OS 권한이 회수됐을 수 있습니다.

해결 절차

  1. OS의 사운드 출력 장치 확인 (PC 스피커/헤드폰 등). 다른 출력으로 잘못 라우팅됐을 수 있어요.
  2. Tikke 설정 → TTS에서 "출력 장치" 드롭다운을 의도한 장치로 변경.
  3. macOS의 경우 시스템 설정 → 개인정보 보호 → 마이크/입력 항목에서 Tikke 권한 확인.
  4. TTS 음량 슬라이더가 0으로 내려가 있지 않은지 확인.

7. 선물 알림이 가끔 누락된다

주요 원인

알림 임계값이 너무 높게 설정됐거나, 같은 시점에 다른 알림이 동시에 들어와 한 개만 표시됐을 수 있습니다.

해결 절차

  1. 알림 임계값을 일시적으로 1코인으로 낮춰 모든 선물이 잡히는지 확인.
  2. 알림 큐 깊이를 늘려(기본 3개 → 10개) 동시 다수 선물에도 모두 표시되도록 변경.
  3. 여전히 누락된다면 TikTok 측 이벤트 발행 지연일 수 있으므로 1~2분 대기 후 재확인.

8. 앱이 한 번씩 갑자기 멈춘다 (응답 없음)

주요 원인

로컬에 쌓인 채팅 로그가 너무 많거나, 메모리가 부족한 상태에서 다른 무거운 앱이 같이 실행됐을 가능성이 있습니다.

해결 절차

  1. 설정 → 데이터 → "오래된 채팅 로그 정리"로 30일 이전 로그 삭제.
  2. OBS·게임·브라우저 등 메모리 큰 앱을 함께 쓰는 경우, Tikke만 다른 화면(다른 작업 공간/모니터)으로 옮기기.
  3. 같은 문제가 반복되면 충돌 리포트(설정 → 진단 → 진단 정보 보내기)를 제출해 주세요.

9. 통계 대시보드 숫자가 갱신되지 않는다

주요 원인

대시보드는 5초 주기로 집계되므로, 너무 짧은 변동은 즉시 반영되지 않을 수 있습니다.

해결 절차

  1. 5~10초 대기 후 자동 갱신 확인.
  2. 여전히 멈춰 있으면 대시보드 우상단의 "새로고침" 버튼.
  3. 차트 시간 범위가 너무 길게 설정됐다면 "최근 15분" 등 짧은 범위로 좁혀 보세요.

10. 앱 업데이트 후 설정이 사라졌다

주요 원인

업데이트 중 설정 마이그레이션이 실패한 경우가 드물게 있습니다.

해결 절차

  1. 설정 → 백업 → "최근 백업에서 복원" 시도.
  2. 클라우드 동기화를 켜둔 상태였다면 클라우드에서 설정 복원 가능.
  3. 복원이 실패하면 충돌 리포트와 함께 문의 채널로 연락 주시면 수동 복구를 도와드립니다.

그래도 해결되지 않으면

위 단계로 해결되지 않는 경우, 다음 정보를 함께 보내 문의 채널로 알려주세요.

  • 운영체제 및 버전 (예: Windows 11 24H2)
  • Tikke 버전 (앱 우측 하단)
  • 문제가 발생한 시점에 사용 중이던 기능
  • 설정 → 진단 → "진단 정보 복사"로 복사한 텍스트
  • 가능하면 짧은 화면 녹화 또는 스크린샷
