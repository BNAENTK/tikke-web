메뉴판 설정
가이드

방송 화면 메뉴판 셀에 선물 매핑, 색상/폰트 적용, 셀 활성화까지 영상으로 안내합니다.

형식 영상 난이도 입문

▶ 재생 버튼을 누르세요 · 전체화면 지원

요약

  1. 사이드바 → 메뉴판 설정 이동.
  2. 셀에 선물 종류 + 표시 텍스트 매핑.
  3. 색상/폰트 크기/굵기 일괄 적용 가능.
  4. URL 복사 → OBS 브라우저 소스 추가.
  5. 설정 내보내기/불러오기로 JSON 백업·복원.

참고

  • 자동 동기화 — 메뉴판 변경 시 클라우드 자동 저장.
  • 그레이스케일 — 미활성 셀 흑백 처리 옵션.
  • JSON 호환 — 다른 PC에 그대로 옮겨 사용 가능.
