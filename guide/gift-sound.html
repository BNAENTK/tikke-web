선물 사운드
등록 가이드

TikTok LIVE에서 받는 선물에 자동 재생할 사운드를 등록하는 방법을 영상으로 안내합니다.

형식 영상 난이도 입문

▶ 재생 버튼을 누르세요 · 전체화면 지원

요약

  1. Tikke 앱 사이드바 → 선물 사운드 이동.
  2. + 파일 추가로 mp3/wav 사운드 파일 업로드.
  3. + 선물 추가로 선물 종류 + 사운드 매핑 규칙 생성.
  4. 라이브 중 해당 선물 수신 시 자동 재생.

참고

  • 동시 재생 ON — 여러 선물 소리 겹쳐서 재생. 동시 재생 한도 초과 시 드랍.
  • 동시 재생 OFF — 순서대로 큐 재생. 대기열 한도 초과 시 드랍.
  • 이모티콘 중복방지 — 100ms 같은 짧은 윈도우 내 연속 이모티콘은 첫 번째만 재생.
