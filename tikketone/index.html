방송을 위한
최고의 TTS 엔진

01
CosyVoice 2 한국어 특화
알리바바 CosyVoice 2 기반 제로샷 보이스 클로닝. 5~10초 음성 샘플만으로 내 목소리를 완벽하게 재현합니다.
02
GPU 로컬 추론 — 완전 오프라인
클라우드 없이 내 PC의 NVIDIA GPU로 직접 추론. 개인정보 보호, 지연시간 최소화, 월 구독 없음.
03
Tikke 앱과 원클릭 연동
Tikke 앱에서 TTS 엔진을 Tikketone으로 선택하면 끝. 별도 설정 없이 실시간 채팅 읽기가 시작됩니다.
04
실시간 채팅 TTS
채팅이 오는 즉시 처리. 큐 기반 자동 재생으로 방송 중 모든 채팅을 내 목소리로 자연스럽게 읽어드립니다.
사용 방법

3단계로 시작

01
목소리 등록
5~10초 음성을 녹음하여 업로드. Tikketone이 자동으로 클로닝 모델 생성.
02
엔진 초기화
Tikketone 앱에서 CosyVoice 2 엔진을 선택하고 초기화. 모델 다운로드 ~1.5GB.
03
Tikke 연동
Tikke 앱 TTS 설정에서 Tikketone 선택. 이제 채팅이 내 목소리로 읽힙니다.
