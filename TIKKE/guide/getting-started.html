처음 설치하고
첫 라이브에 적용하기

Tikke는 휴대폰 한 대로 라이브를 송출하는 흔한 TikTok LIVE 환경에 한 대의 노트북을 옆에 두는 구성을 가정해 만들어졌습니다. 이 글에서는 설치부터 첫 라이브에 적용하기까지 30분 안에 끝낼 수 있는 순서를 안내합니다.

예상 소요 30분 난이도 입문 대상 OS Windows · macOS

시작 전에 준비물 두 가지를 확인하세요. ① 인터넷이 잘 되는 노트북 한 대(Windows 10/11 또는 macOS), ② 평소 라이브를 켜는 TikTok 계정. 휴대폰은 평소처럼 라이브 송출용으로 쓰시면 됩니다.

이 글의 흐름
  1. 설치 — 운영체제별 다운로드
  2. 첫 실행 — 환경 자동 점검
  3. TikTok 계정 연결 — 한 번이면 끝
  4. 라이브 연결 — 한 번 클릭으로 도킹
  5. 처음 만져야 할 설정 다섯 가지
  6. 첫 라이브에서 확인할 체크리스트

1. 설치 — 운영체제별 다운로드

메인 페이지 다운로드 섹션으로 가서 자신의 OS에 맞는 빌드를 받습니다. Windows 10/11(64-bit)과 macOS Apple Silicon · Intel을 모두 지원합니다.

  • Windows: Tikke-Setup.exe를 받아 더블클릭하면 인스톨러가 실행됩니다. 설치 위치는 기본값을 권장합니다.
  • macOS: Tikke.dmg를 열고 Tikke 아이콘을 Applications 폴더로 드래그합니다. 처음 실행 시 "확인되지 않은 개발자" 경고가 뜨면, 시스템 설정 → 보안 → "그래도 열기"로 허용해주세요.
!

설치 파일은 반드시 공식 도메인에서. Tikke의 유일한 공식 배포처는 tikke.kr입니다. 다른 곳에서 받은 파일은 변조됐을 수 있어요.

2. 첫 실행 — 환경 자동 점검

처음 Tikke를 실행하면 5~10초 정도 환경 점검 화면이 뜹니다. 이 단계에서 Tikke는 다음을 확인합니다.

  • 네트워크 연결 상태 (TikTok 이벤트 스트림 접근 가능 여부)
  • 운영체제 권한 (마이크·오디오 출력 — TTS와 사운드 라이브러리 사용에 필요)
  • 이전에 설치된 버전의 잔여 설정 (있다면 마이그레이션)

점검이 끝나면 빈 컨트롤 룸 화면이 보입니다. 아직 라이브에 연결되지 않은 상태입니다 — 다음 단계로 넘어가요.

3. TikTok 계정 연결 — 한 번이면 끝

상단 우측의 "라이브 연결" 버튼을 누르고 자신의 TikTok ID(@ 제외)를 입력합니다. 예를 들어 @tikke_demo 계정이라면 tikke_demo만 입력합니다. 그러면 Tikke는 다음을 자동으로 처리합니다.

  1. 입력한 핸들이 실제로 존재하는 계정인지 확인
  2. 해당 계정이 라이브를 켜는 순간을 감지할 준비를 시작
  3. 인증 토큰을 운영체제의 안전한 자격 증명 저장소(Windows Credential Manager / macOS Keychain)에 보관

이 작업은 보통 5초 이내에 끝납니다. 한 번 인증해두면, 다음부터는 Tikke를 켤 때마다 자동으로 같은 계정에 연결됩니다.

i

계정 비밀번호는 Tikke에 저장되지 않습니다. Tikke는 TikTok이 발급하는 일회용 인증 토큰만 사용하며, 비밀번호 자체는 받지도 저장하지도 않습니다. 이 점은 개인정보처리방침에도 명시되어 있습니다.

4. 라이브 연결 — 한 번 클릭으로 도킹

입력이 끝나면 컨트롤 룸 좌상단에 "대기 중" 상태가 표시됩니다. 이제 휴대폰으로 TikTok 앱을 열어 라이브를 시작한 뒤, Tikke에서 "라이브 연결" 버튼을 한 번 눌러주세요. 그러면 현재 방송 중인 라이브 세션과 연결되어 채팅·선물·시청자 이벤트가 흐르기 시작합니다.

라이브를 시작하기 전에 연결 버튼을 누르면 "해당 계정이 현재 라이브 중이 아닙니다"라는 메시지가 뜨니, 휴대폰에서 라이브를 먼저 시작하고 Tikke에서 연결 버튼을 누르는 순서로 진행하세요.

5. 처음 만져야 할 설정 다섯 가지

기본값으로도 잘 작동하지만, 한 번만 만지면 라이브 경험이 크게 좋아지는 설정 다섯 가지를 우선 처리하세요.

① 번역 활성화

설정 → 번역 메뉴에서 자동 번역을 켜고, 번역할 원본 언어를 "자동 감지"로, 도착 언어를 "한국어"로 두세요. 외국어 채팅이 들어오면 자동으로 한국어 자막으로 정리되어 표시됩니다.

② 키워드 강조

채팅 → 키워드 강조에 자신의 닉네임, 자주 들어오는 질문 키워드(예: "방송시간", "셋업", "노래")를 등록해 두세요. 빠른 채팅 흐름에서도 해당 단어가 들어오면 색이 바뀌어 눈에 띕니다.

③ 선물 알림 사운드

알림 → 사운드에서 큰 선물에만 알림음이 울리도록 임계값을 정해 두세요. 너무 작은 선물에 모두 사운드가 울리면 방송이 산만해지고, 너무 큰 선물에만 울리면 시청자가 인식하기 어려운 균형이 있어요. 추천 시작값은 50코인입니다.

④ 답장 대기 단축키

특정 채팅에 나중에 답하고 싶다면 그 메시지에 마우스를 올리고 R 키를 누르면 "답장 대기" 목록으로 옮겨집니다. 이 단축키는 빠른 채팅에서 진가를 발휘하니 미리 손에 익혀 두는 게 좋아요.

⑤ TikTok LIVE Studio 오버레이 연결

오버레이 → 새 오버레이에서 원하는 위젯(채팅, 알림, 목표바)을 만들면 임시 URL이 생성됩니다. 이 URL을 TikTok LIVE Studio의 "링크 소스"로 추가하면 방송 화면에 위젯이 표시됩니다. 자세한 디자인 팁은 오버레이 가이드에서 다룹니다.

6. 첫 라이브에서 확인할 체크리스트

이제 진짜 라이브를 시작할 시간입니다. 첫 30분 동안 다음 항목들이 의도대로 작동하는지 가볍게 확인해보세요.

  • 채팅이 실시간으로 흐르고 있는가 (지연 1~2초 이내)
  • 외국어 채팅이 자동으로 한국어로 번역되어 따라붙는가
  • 키워드 강조가 등록한 단어에 반응하는가
  • 선물이 들어오면 사운드와 시각 알림이 동시에 작동하는가
  • 대시보드의 시청자·팔로우 통계가 변화하고 있는가

모두 정상이면 셋업 완료입니다. 다음 단계로 채팅 모더레이션이나 번역 활용 가이드로 넘어가도 좋고, 그냥 평소 라이브를 진행하면서 Tikke가 옆에서 도와주는 환경에 익숙해지는 것도 좋습니다.

다음에 읽으면 좋은 글

  • 빠른 채팅을 놓치지 않는 모더레이션 베스트 프랙티스 — 시청자가 100명 넘어가면서 채팅이 너무 빨라질 때 참고하세요.
  • 실시간 번역으로 글로벌 시청자와 소통하기 — 해외 시청자 비율이 늘기 시작하면 꼭 읽어보세요.
  • 자주 발생하는 문제 해결 — 어딘가에서 막혔다면 먼저 이쪽을 확인하세요.
